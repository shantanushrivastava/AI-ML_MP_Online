# AI-ML Practice Code

A collection of Machine Learning implementations in Python covering supervised and unsupervised learning algorithms, with real-world datasets and visualizations.

## Project Structure
```
PracticeCodeML/
├── SupervisedLearning/
│   ├── LinearRegression.py        # Simple linear regression model
│   └── polynomialRegression.py    # Polynomial regression with plotting
└── UnsupervisedLearning/
    └── K-MeansClustering.py       # K-Means clustering algorithm
```

## Algorithms Covered

**Supervised Learning**
- Linear Regression - Predicting continuous output values
- Polynomial Regression - Non-linear data fitting with matplotlib visualization

**Unsupervised Learning**
- K-Means Clustering - Grouping data points without labeled training data

## Tech Stack

- Python 3.13
- scikit-learn
- matplotlib
- numpy

## Getting Started
```bash
pip install scikit-learn matplotlib numpy
```

Run any script directly:
```bash
python SupervisedLearning/LinearRegression.py
python SupervisedLearning/polynomialRegression.py
python UnsupervisedLearning/K-MeansClustering.py
```
